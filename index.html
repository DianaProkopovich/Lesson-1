Команды Git

  • git clone path_to_repository (https)
  • git status статус текущего локального репозитория, указывает какие изменения произошли в файле
  • git add path_to_file - Добавить новый файл под контроль GIT-a
  • git commit --am 'added a new file index.html' - фиксирует изменения, добавляет комментарии
  • git push - отправить ЗАКОМИЧЕННЫЕ изменения на сервер jjj
  • git pull - забрать обновления
